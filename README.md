### About Me

I'm a software engineer with 5+ years building secure, production-grade backend and cloud systems — from large-scale AWS serverless infrastructure to EVM-based marketplaces and smart contracts. I've spent recent years as the sole engineer leading an Infrastructure-as-Code migration for a live production backend, owning 50–75 Lambda functions and their APIs end to end, alongside independent security auditing across the stack.

Lately I've been extending that systems background into **applied AI/LLM engineering** — building with Python, FastAPI, retrieval-augmented generation (RAG), and agent-based architectures.

I learn fast, ask a lot of "why does this actually work" questions, and I'm comfortable owning something I've never touched before.
